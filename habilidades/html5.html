¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado HTML, utilizado para estructurar y presentar contenido en la web. Es un estándar fundamental para el desarrollo web moderno.

Características Principales

  • Estructura semántica mejorada
  • Soporte nativo para audio y video
  • Canvas para gráficos
  • Almacenamiento local
  • Formularios mejorados

Ejemplo de Código

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página</title>
</head>
<body>
    <header>
        <h1>Título Principal</h1>
    </header>
    <main>
        <p>Contenido de la página</p>
    </main>
</body>
</html>

Ventajas de HTML5

  • Mejor compatibilidad con navegadores
  • Mayor accesibilidad
  • Mejor rendimiento
  • Estructura más clara y semántica
